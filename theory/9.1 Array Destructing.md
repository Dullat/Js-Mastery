
## üß† What is Array Destructuring?

**Array destructuring** is a syntax that lets you **unpack values from arrays** (or iterable objects) into individual variables, in a clean and concise way.

---

## ‚úÖ Basic Syntax

```js
const [a, b] = [1, 2];
console.log(a); // 1
console.log(b); // 2
```

Think of it like saying:

> "Assign the 1st value of the array to `a`, the 2nd to `b`."

---

## üìö Examples

### 1. **Simple Destructuring**

```js
const [x, y, z] = [10, 20, 30];
console.log(x); // 10
console.log(y); // 20
console.log(z); // 30
```

---

### 2. **Skipping Elements**

```js
const [first, , third] = ['a', 'b', 'c'];
console.log(first); // 'a'
console.log(third); // 'c'
```

---

### 3. **Default Values**

```js
const [a = 1, b = 2] = [10];
console.log(a); // 10
console.log(b); // 2 (default used)
```

---

### 4. **Swapping Variables**

```js
let a = 1, b = 2;
[a, b] = [b, a];
console.log(a); // 2
console.log(b); // 1
```

---

### 5. **Rest Operator (`...`)**

```js
const [head, ...tail] = [1, 2, 3, 4];
console.log(head); // 1
console.log(tail); // [2, 3, 4]
```

---

### 6. **Nested Destructuring**

```js
const [a, [b, c]] = [1, [2, 3]];
console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
```

---

### 7. **Destructuring in Function Parameters**

```js
function printCoords([x, y]) {
  console.log(`X: ${x}, Y: ${y}`);
}

printCoords([5, 10]); // X: 5, Y: 10
```

---

### üîÅ Use Case: With `Object.entries()`

```js
const user = { name: "Ali", age: 25 };

for (const [key, value] of Object.entries(user)) {
  console.log(`${key}: ${value}`);
}
// name: Ali
// age: 25
```

## ‚ùå Common Mistakes

|Mistake|Why It Fails|
|---|---|
|`[a, b] = 10`|RHS must be iterable|
|`const [a, b] = null`|`null` is not iterable|
|Forgetting `let`/`const`|Leads to ReferenceError if variables aren‚Äôt declared|
